# `cli`

> Apids command line tool.

## Usage

```bash
## generate Api spec files
apids g rest

## generate Typescript models
apids g types

## generate Fastify server files
apids g fastify

## generate all artifacts in one go (api spec, types and server files)
apisds g
```
