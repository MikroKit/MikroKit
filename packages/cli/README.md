# `cli`

> mion command line tool.

## `CLI`

```shell
## generate Open Api spec files
npx mion g openApi

## generate Api browser client (typescript)
npx mion g apiClient


## generate all artifacts in one go (api spec, types and server files)
npx mion g
```
