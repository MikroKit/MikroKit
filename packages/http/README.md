# `@mionkit/http`

**mion is an RPC like router oriented for quick Api development,** it is agnostic about the environment it is used on. It could be used on [serverless environments](../serverless/README.md), or as an standalone http server. it offers quick development, fast execution and a Lightweight router. [Benchmarks here!](https://github.com/MionKit/benchmarks) ðŸš€

This is the [Node.js](https://nodejs.org/en) http server for mion, only supports `json` content type, does not support `multipart/form-data`, no websocket or streams and no file upload neither. There are better alternatives for those scenarios (like S3 file upload, etc).
